# 🌼 Daffodils Infant School Website

> Quality early childhood education in Patna, Bihar since 1999

A modern, responsive website for Daffodils Infant School built with pure HTML, CSS, and JavaScript.

## 🚀 Features

- **Responsive Design** - Works perfectly on all devices
- **Modern Animations** - Smooth scroll animations and interactions
- **Photo Gallery** - Filterable gallery with lightbox
- **Interactive Contact** - Contact form with map integration
- **Professional UI** - Modern design with gold and teal theme
- **SEO Optimized** - Meta tags and semantic HTML
- **Fast Loading** - Optimized images and minimal dependencies

## 🌐 Deployment

### GitHub Pages (Free Hosting)
1. Push to GitHub repository
2. Go to Settings → Pages
3. Select source as "Deploy from a branch"
4. Choose main branch and root folder
5. Save and wait for deployment

### Other Hosting Options
- Netlify
- Vercel
- Any static hosting service
